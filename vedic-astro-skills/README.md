# Vedic Astro Skills

A small starter package for Vedic astrology utilities (stubs and examples).

This directory contains a minimal Python package skeleton to hold skills, utilities, and examples for working with Vedic astrology calculations. The implementations are intentionally minimal and provided as a starting point — replace the placeholder algorithms with authoritative astronomical/astrological libraries if you need production accuracy.

Contents
- vedic-astro-skills/__init__.py — package metadata
- vedic-astro-skills/skills.py — core module skeleton with stubbed methods
- vedic-astro-skills/example_usage.py — a short example showing how to instantiate and call the API

Installation

You can copy the vedic-astro-skills directory into a Python project, or extend it into a pip-installable package.
